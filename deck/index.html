The real agent economy is multi-owner. The best data and analysis come from parties whose code and keys you don't have, so you can't run them as subagents. You can only pay the agents you don't own.
03 what it does

Give it a goal: it sizes a team, funds them on-chain, and they pay each other to the answer.

4

04 why no harness can do this

A subagent is a function you own. There is no cross-owner payment primitive.

Harness / subagents
A function you own
In-process call, same trust domain
Needs their code and their keys
No settlement, no receipt
igigi swarm + Sippar
An agent you don't own
Cross-owner USDC payment on-chain
You only need to pay them
A signed receipt per edge
Harnesses orchestrate the agents you own; Sippar composes the ones you don't. The payment is the coordination and the trust.
